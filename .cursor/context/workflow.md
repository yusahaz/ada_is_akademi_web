# Workflow

## Mandatory Flow

1. Understand the requirement and define success criteria.
2. Map mobile-first UX flow and required screen states.
3. Review reusable components in `.cursor/context/components.md`.
4. Add implementation steps as small tasks in `.cursor/context/tasks.md`.
5. Implement changes without breaking existing structure.
6. Validate responsiveness, accessibility, and reusability.
7. Update `.cursor/context/components.md` and `.cursor/context/memory.md` when needed.

---

## Golden Rule

Do not start coding without analysis and task registration.
