# Design System

## Design Philosophy

- Clean
- Modern
- Minimal but expressive
- Mobile-first always

---

## Spacing

- 8pt grid is the baseline system
- Spacing scale: 4 / 8 / 12 / 16 / 24 / 32 / 48 / 64
- All layouts must align to this scale

---

## Typography

- Headings: bold, clear hierarchy
- Body: readable, medium weight
- Text sizes are selected for mobile readability (`sp` intent)

---

## Colors

Primary: #4F46E5
Secondary: #22C55E
Neutral: gray scale

---

## Responsiveness

- Design for mobile (small viewport) first, then scale up.
- Maintain visual balance consistency across iOS and Android.
- Use safe-area-aware layout patterns (no clipping at top/bottom edges).

---

## UI Patterns

- Cards with soft shadows
- Rounded corners (2xl)
- Smooth transitions
- Vector icons are required; avoid raster assets unless necessary

---

## Animations

- Fast (150-300ms)
- Ease-in-out
- Motion should be purposeful and non-distracting

---

## UX Rules

- Always clear CTA
- Avoid clutter
- Keep hierarchy obvious
- Provide explicit feedback and error states in forms

---

## Accessibility

- Contrast must be strong
- Buttons must be clear
- Touch targets must be large enough for mobile interaction
